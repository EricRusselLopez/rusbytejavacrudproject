# 🚀 RusByte Net - User Management System

**👨‍💻 Developer:** Eric Russel M. Lopez

**This RusByte Net - User Management System** is an extension of my **RusByte Net project**, I've applied my skills in security, and UI development. This Java application provides a secure and efficient user authentication system.

This system allows administrators to **register, authenticate, and manage users efficiently** while providing a seamless user experience with interactive popups, loading indicators, and restricted access based on authentication status.

## ✨ Current Features

- 🔒 **Secure User Authentication** (Email address & Password)
- 📌 **App Router** (Switch between pages dynamically) -> Like ReactJs Router
- 📌 **Persistent User Session** (Keeps users logged in) -> Maintains the user's logged-in state by storing session data in a file, similar to how **localStorage and Cookies** (in web) work, until the user logs out.
- 📌 **Create, Read, Update, and Delete users** (CRUD) -> Manage users
